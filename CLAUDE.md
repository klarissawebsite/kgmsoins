# Claude Instructions: KGM Soins

This repository contains the KGM Soins public website.

Use the Aurex-derived animation stack responsibly:

- Keep the UI calm, modern, and appropriate for home nursing and accompaniment.
- Keep French copy clear and reassuring.
- Never commit API keys, database passwords, or service-role secrets.
- Store editable assets in `public/assets`.
- Validate responsive layouts before deployment.
